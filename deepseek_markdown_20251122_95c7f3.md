# لحظة فن - Art Moment

نظام إدارة متكامل لاستوديوهات الطباعة الفوتوغرافية وإدارة الطلبات الفنية.

## المميزات
- لوحة تحكم شاملة بالإحصائيات
- إدارة كاملة للطلبات والعملاء
- متابعة المدفوعات والحالات
- واجهة مستخدم عربية 100%

## طريقة النشر على Netlify

1. انشئ مستودع جديد على GitHub وأضف كل الملفات
2. اذهب إلى [netlify.com](https://netlify.com)
3. انقر على "Add new site" واختر "Import an existing project"
4. اختر المستودع الذي أنشأته
5. انقر "Deploy site"

## التطوير المستقبلي

- إضافة نظام تسجيل الدخول
- رفع الصور وتخزينها
- إشعارات عبر البريد الإلكتروني
- تقارير متقدمة
