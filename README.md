# 🎨 لحظة فن - Art Moment

نظام إدارة متكامل لاستوديوهات الطباعة الفوتوغرافية وإدارة الطلبات الفنية.

## 🌟 المميزات

- 📊 لوحة تحكم شاملة بالإحصائيات
- 🛒 إدارة كاملة للطلبات والعملاء
- 💳 متابعة المدفوعات والحالات
- 🎯 واجهة مستخدم عربية 100%
- 📱 تصميم متجاوب لجميع الأجهزة

## 🚀 النشر على Vercel

1. انشئ مستودع جديد على GitHub وأضف كل الملفات
2. اذهب إلى [vercel.com](https://vercel.com)
3. انقر على "Add New Project" واختر "Import Git Repository"
4. اختر المستودع الذي أنشأته
5. انقر "Deploy"
