# PrintFlow - نظام إدارة طباعة الصور

نظام متكامل لإدارة مشاريع طباعة الصور الفوتوغرافية، يدعم تتبع الطلبات والعملاء والمدفوعات.

## المميزات

- 📊 لوحة تحكم شاملة
- 📝 إدارة الطلبات وتتبع حالتها
- 👥 قاعدة بيانات العملاء
- 💰 نظام متابعة المدفوعات
- 🔍 إمكانيات البحث والتصفية
- 📱 تصميم متجاوب

## التقنيات المستخدمة

- React
- React Router
- Tailwind CSS
- Lucide React Icons
- Date-fns

## طريقة التشغيل

1. انسخ الملفات إلى أي خدمة استضافة ثابتة (Netlify, Vercel, GitHub Pages)
2. لا حاجة لتثبيت أي dependencies - كل المكتبات مستضافة على CDN
3. افتح `index.html` في المتصفح

## التطوير المستقبلي

- إضافة نموذج إدخال البيانات
- رفع الصور وتخزينها
- إشعارات عبر البريد الإلكتروني
- تقارير متقدمة
- نسخة احتياطية للبيانات
